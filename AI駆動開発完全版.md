特徴
説明
無料・オープンソース

豊富なステンシル

ダウンロード: https://pencil.evolus.vn/
インストール手順
# Mac (Homebrew)
brew install --cask pencil

# Windows
# 公式サイトからインストーラーをダウンロード

# Linux (Ubuntu/Debian)
# 公式サイトから.debファイルをダウンロード

やること
各画面の構成要素と配置をPencilで作成する。
Pencilでの作業手順
1. 新規ドキュメント作成
   File → New Document

2. ステンシルを選択
   左パネルから使用するUI部品を選択
   - "Common Shapes" - 基本図形
   - "Android" - Androidアプリ用
   - "iOS" - iOSアプリ用  
   - "Sketchy GUI" - 手書き風（Lo-Fi向け）

3. 画面を作成
   - 部品をドラッグ&ドロップで配置
   - テキストをダブルクリックで編集
   - プロパティパネルでサイズ・色を調整

4. ページを追加
   - Document → New Page で画面を追加
   - ページ名を「01-ホーム」「02-検索」などにする

5. 画面間リンクを設定（オプション）
   - ボタンを選択 → プロパティ → Link To
   - リンク先のページを選択

6. エクスポート
   - Document → Export Document
   - PNG（画像）またはPDF（ドキュメント）で出力

🎯 成功のための心得
Do（やるべきこと）
要件定義に時間をかける - 全体の30%以上
小さく分割する - 1タスク30分〜2時間
検証可能にする - テスト、スクリーンショット
早めに軌道修正 - 2回同じミス→ /clear
ドキュメントを残す - 未来の自分のために
Don't（避けるべきこと）
「全部作って」と言わない - 必ず分割
曖昧な指示をしない - 具体的に
暴走を放置しない - Escで止める
テストを省略しない - 後で苦労する
デザインを飛ばさない - 手戻りの原因
